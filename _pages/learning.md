在 iTron，我們深信「做中學」是最好的成長方式。我們為每位新加入的成員提供系統性的教學與訓練，無論你來自哪個科系、有沒有相關經驗，只要你對機器人充滿熱情，都歡迎你加入我們，一同學習、成長。

### 我們的教學規劃

為了幫助新成員快速融入團隊並掌握核心技能，我們規劃了以下的學習路徑：

1.  **新生入門教學**
    在學期初，我們會有一系列的入門課程，由資深社員親自授課，內容涵蓋軟體與機構的基礎知識，讓你對機器人開發有全面的認識。

2.  **分組專業訓練**
    完成入門教學後，成員可以根據自己的興趣選擇加入軟體或機構組，進行更深入的專業技能訓練。各組會由學長姐帶領，透過小組討論和實作練習，深化學習效果。

3.  **專案實戰**
    團隊的核心精神是「從做中學」。你將有機會參與實際的機器人專案，將所學知識應用於解決真實世界的問題。從設計、開發到測試，你將完整體驗一個專案的生命週期。

### 你將學到什麼？

根據你的興趣與選擇的分組，你將有機會學習到以下專業技能：

#### 軟體組
- **程式語言:** C++, Python
- **機器人作業系統:** ROS/ROS2
- **版本控制:** Git, GitHub
- **開發環境:** Linux, Docker
- **演算法:** 電腦視覺、路徑規劃、運動學

#### 硬體組
- **電腦輔助設計 (CAD):** SolidWorks、Blender
- **電腦輔助製造 (CAM):** 3D列印、CNC加工、雷射切割
- **材料力學與結構分析**
- **公差與配合**

我們鼓勵成員跨領域學習，無論你隸屬哪個組別，都有機會接觸到其他領域的知識，成為全方位的機器人開發人才。
